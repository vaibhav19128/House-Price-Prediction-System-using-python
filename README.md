# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

This project is part of the IBM Data Science Professional Certificate. The objective is to build a machine learning model that predicts house prices based on various property features. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation using regression techniques.

---

## 📂 Repository Structure

```
├── Final_Project_IBM.ipynb      # Jupyter Notebook containing the complete project
├── kc_house_data_NaN.csv        # Dataset used for training and testing
├── README.md                    # Project documentation
```

---

## 🎯 Objectives

- Import and explore the dataset
- Clean missing and duplicate values
- Perform Exploratory Data Analysis (EDA)
- Analyze feature relationships

- Evaluate model performance
- Predict house prices accurately
